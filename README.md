# あとで-Labo

流れていく情報、予定、あとで見たいものを一時的に置いておくための小さなブラウザアプリです。

## できること

- タイトルだけで即保存
- カテゴリ、優先度、状態、予定日時、締切、メモの登録
- 今日見るもの、近日予定、締切が近いもの、あとで見るもの、未対応一覧の表示
- 完了、編集、削除
- カテゴリ、優先度、今日、今週、期限ありの絞り込み
- localStorage保存
- JSONエクスポート、JSONインポート

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```
